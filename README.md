## ✨ Sobre mim

Aspirante a **Frontend Engineer** cursando **ADS na FIAP**. Adoro transformar ideias em experiências elegantes e acessíveis. Trabalho com **React/Expo** no front, **Python** no back (APIs) e **OracleDB/SQL** nos dados. Priorizo **design responsivo**, **performance**, **componentização** e **boas práticas** (lint, testes e CI/CD).

* ⚛️ React + Hooks/Context, roteamento, gerenciamento de estado
* 📱 Expo/React Native para apps iOS/Android
* 🐍 Python (FastAPI/Flask) para APIs e automações
* 🗄️ OracleDB/SQL (queries otimizadas, procedures, modelagem)
* 🧪 Testes, Docker, GitHub Actions e organização de monorepo
